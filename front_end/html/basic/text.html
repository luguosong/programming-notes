文本标签

em标签

em标签：斜体，要着重阅读的内容

strong标签

strong标签：加粗，十分重要的内容（语气比em要强）

span标签

span标签：没有语义，用于包裹短语的通用容器。用于设置文本的样式，或者为文本设置id

cite标签

cite标签：引用，引用的作品名称

dfn标签

dfn标签：定义，定义术语，专属名词

del标签

del标签： 删除线，表示文本被删除

ins标签

ins标签： 下划线，表示文本被插入

sub标签

sub标签：下标，表示文本是下标,比如水：H2O

sup标签

sup标签：上标，表示文本是上标，8的3次方：83

code标签

code标签：System.out.println("hello world");

kbd标签

kbd标签：表示键盘文本，Ctrl+Alt+Del

abbr标签

abbr标签：表示缩写，HTML

bdo标签

bdo标签：表示文本方向，从右到左的文本

var标签

var标签：表示变量，x = y + 2

small标签

small标签：將使文本的字体变小一号。(例如从大变成中等，从中等变成小，从小变成超小)。在 HTML5 中，除了它的样式含义，这个元素被重新定义为表示边注释和附属细则，包括版权和法律文本。--很少使用

b标签

b标签：用于吸引读者的注意到该元素的内容上（如果没有另加特别强调）。这个元素过去被认为是粗体（Boldface）元素，并且大多数浏览器仍然将文字显示为粗体。尽管如此，你不应将 b 元素用于显示粗体文字；替代方案是使用 CSS font-weight 属性来创建粗体文字。

i标签

i标签：用于表现因某些原因需要区分普通文本的一系列文本。例如技术术语、外文短语或是小说中人物的思想活动等，它的内容通常以斜体显示。现在多用于字体图标

u标签

u标签：与正常内容反差文本。例如：错的单词、不合适的描述等。默认情况下渲染为一个实线下划线，可以用 CSS 替换。

q标签

q标签：短的引用。浏览器通常会在引用的内容前后包裹引号。

blockquote标签

blockquote标签：
长的引用。通常在引用的内容前后添加换行，以及缩进；也可以通过 CSS 来改变样式。

address标签

address标签：
联系信息。通常呈现为斜体。浏览器通常会在引用的内容前后包裹引号。
