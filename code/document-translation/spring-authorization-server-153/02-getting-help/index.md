# 获取帮助

## 社区

欢迎加入 [Spring Security 社区](https://docs.spring.io/spring-security/reference/community.html)。Spring Authorization Server 是由 Spring Security 团队主导的一个开源项目。如果你在使用 Spring Authorization Server 时遇到问题，我们随时乐意提供帮助。

## 资源

以下是获取帮助的一些最佳途径：

- 先看看 [How-to 指南](https://docs.spring.io/spring-authorization-server/reference/how-to.html)，其中给出了对最常见问题的解决方案。
- 学习 Spring Authorization Server 所依赖的 Spring Security 基础知识。如果你刚开始接触 Spring Security，可以查阅[参考文档](https://spring.io/projects/spring-security#learn)，或者运行一些[示例项目](https://github.com/spring-projects/spring-security-samples)。
- 通读本篇文档。
- 尝试我们的各类[示例应用程序](https://github.com/spring-projects/spring-authorization-server/tree/main/samples)。
- 在 Stack Overflow 上使用[spring-authorization-server标签](https://stackoverflow.com/questions/tagged/spring-authorization-server)提问。
- 在 [GitHub](https://github.com/spring-projects/spring-authorization-server/issues) 上提交 Bug 报告和功能增强请求。

!!! note

	Spring Authorization Server 是一个开源项目，其中也包括文档。如果你在文档中发现问题，或者希望对其进行改进，欢迎[参与贡献](https://github.com/spring-projects/spring-authorization-server)。
