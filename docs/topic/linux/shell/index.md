# Shell 与文本处理

Linux 系统管理的日常工作中，大部分时间都花在 Shell 命令行和文本操作上。无论是编写自动化脚本、分析日志文件、还是编辑配置，都需要掌握 Shell 语言和文本处理工具链。

本专题从编辑器、Shell 语言基础到文本处理工具，逐步覆盖日常最常用的技能：

- **Vim 编辑器**：Linux 上最经典的终端文本编辑器，掌握基本操作后编辑效率远超鼠标
- **Bash Shell 基础**：变量、环境配置、别名、历史命令等 Shell 日常用法
- **重定向与管道**：理解数据流的方向控制，学会用管道组合命令
- **正则表达式**：文本匹配的通用语法，是 `grep`、`sed`、`awk` 等工具的基础
- **文本处理工具**：`grep`、`sed`、`awk`、`cut`、`sort`、`uniq`、`tr` 等常用工具
- **Shell 脚本编程**：将日常操作封装为可复用的脚本，条件判断、循环、函数
