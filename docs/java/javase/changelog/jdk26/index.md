# JDK 26 更新日志

JDK 26 各更新版本（26.0.X）的变更记录，按发布时间倒序排列。数据来源于 [Oracle JDK 26 Release Notes](https://www.oracle.com/java/technologies/javase/26u-relnotes.html)。

> JDK 26 为当前最新非 LTS 版本（2026 年 3 月发布），下一个非 LTS 版本发布后即进入延伸支持。

## 26.0.1（2026-04-21）

### 变更与增强

- 包含 IANA 时区数据 2026a
- 符合 Java SE 26 规范（JSR 401，2026-03-17）
- FreeType 升级至 2.14.2

### 安全修复

- 修复多个安全漏洞（详见 [Oracle Critical Patch Update April 2026](https://www.oracle.com/security-alerts/cpuapr2026.html)）

### Bug 修复

- 完整修复列表详见 [JDK 26.0.1 Release Notes](https://www.oracle.com/java/technologies/javase/26-0-1-relnotes.html)
